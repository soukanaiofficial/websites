SOU KANAI SCATTER



“Sou Kanai Scatter” makes it possible to randomly duplicate and array selected layers with functions to randomize their properties in the composition.

Functions:

Size Random (0-100%)
Rotation (Z) Random (0-360°)
Opacity Random (0-100%)
Scatter Amount (0-100)

To run “Sou Kanai Scatter”, please go to Adobe After Effects > Preferences > General, check on “Allow Scripts to Write Files and Access Network”, go to Scripts > Run Script File, and open “Sou Kanai Scatter.jsx”.

The script version is regularly updated.
